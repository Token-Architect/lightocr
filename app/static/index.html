支持格式说明

📄 文档与图片识别

  • PDF: 原生支持，自动分页识别
  • Office: .doc, .docx, .ppt, .pptx (自动转PDF后识别)
  • 图片: .jpg, .png, .jpeg, .bmp
  • 其他: .txt, .md, .xls, .xlsx (直接文本提取)

🖼️ 智能子图提取

对于包含复杂图表、公式或手绘草图的文件，系统会自动：

  • 检测并裁剪出独立子图
  • 保存为高清图片资源
  • 在 Markdown 结果中自动插入图片链接
  • 支持物理题、数学公式、工程图纸等场景

API 接口说明

POST /ocr/submit

提交 OCR 识别任务

curl -X POST "{{ SERVICE_BASE_URL }}/ocr/submit" \
  -F "uuid=task-123456" \
  -F "file=@/path/to/your/image.png" \
  -F "ocr_method=rapiddoc" # 可选: rapiddoc (默认) 或 vlm_opencv

响应示例 (Response):

{
  "task_id": "task-123456",
  "status": "pending"
}
GET /ocr/status/{task_id}

查询任务处理状态

curl "{{ SERVICE_BASE_URL }}/ocr/status/task-123456"

响应示例 (Response):

{
  "task_id": "task-123456",
  "status": "processing"
}
GET /ocr/result/{task_id}

获取识别结果（Markdown 格式）

curl "{{ SERVICE_BASE_URL }}/ocr/result/task-123456"

响应示例 (Response):

{
  "task_id": "task-123456",
  "status": "completed",
  "filename": "image.png",
  "content": "# 标题\n\n正文内容...\n\n![diagram](...)\n\n",
  "duration": 2.5
}
